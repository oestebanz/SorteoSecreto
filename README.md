# SorteoSecreto

Tarea integradora del proyecto SorteoSecreto en Java - Sistema de gestion de sorteos de Amigo Secreto.

## Entrega 2 - Programacion Orientada a Objetos

Implementacion con separacion de responsabilidades en paquetes model y ui, aplicando POO.

## Estructura del proyecto

- src/model/ : Clases del modelo de dominio
- src/ui/ : Interfaz de usuario por consola
- bin/ : Archivos .class compilados
- docs/API/ : Javadoc generado
- docs/ClassDiagram.pdf : Diagrama de clases UML
- docs/SorteoSecreto_E2.vpp : Proyecto Visual Paradigm
- legacy/ : Codigo de la Entrega 1 (referencia)

## Requerimientos

- Java 17 o superior
